Count Numbers Task

In this task, participants had 1.5 minutes to solve as many questions as possible. In each question, they had to count the number of ones in a 5x5 matrix.
In the example below the correct answer is 12 since there are twelve 1s in the matrix.

Each correctly solved question earned a point.

How many more questions do you think men (women) answered correctly, compared to women (men)? [in percentage points]
